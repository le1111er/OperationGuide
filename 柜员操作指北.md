# 柜员操作手册

## 存取款

**金额 >= 50000 || 代理人 >= 10000 || 未成年人 >= 5000 需提供身份证，未成年人额外提供户口本（户主页，监护人页，本人页）**

1. **011003** 活期存款支取（存钱）

   ``` mermaid
   graph LR
   A[刷卡] --> B{是否本人？}
   B --> |是| C(签字或手印)
   B --> |否| D(代理人身份信息)
   D --> |签代理人| E(拍照)
   E --> C
   C --> |按手印有人像材料| F[给钱]
   ```

2. 存钱

##  代缴

**121013** 单笔实时缴费

1.  医社保 代码309 两险**
